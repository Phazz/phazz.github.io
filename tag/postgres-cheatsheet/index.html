Postgres cheatsheet

Shell command starts with $ Postgres command starts with # List databases from shell: $ psql -l Connect to the database postgres: $ psql postgres Get postgres command help: # \? Get all users: # \du Create a user and set a password: $ sudo -u postgres psql postgres # \password postgres Create a database: # create database my_new…
