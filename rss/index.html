]]>http://my-ghost-blog.com/postgres/93f852ea-0baa-4c7d-858b-2b8a16edd735Tue, 29 Apr 2014 10:10:53 GMT<![CDATA[Postgres cheatsheet]]>Shell command starts with $
Postgres command starts with #

List databases from shell: 

$ psql -l

Connect to the database postgres: 

$ psql postgres

Get postgres command help: 

# \?

Get all users: 

# \du

Create a user and set a password: 

$ sudo -u postgres psql postgres
# \password postgres

Create a database: 

# create database my_new_db;

Connect to database my_db: 

\c my_db

Copy result of a request from a client to csv: 

$ \copy (Select * From foo) To '/tmp/test.csv' With CSV

On heroku: 

psql "dbname=foo host=bar.compute-1.amazonaws.com user=abc password=123 port=5432 sslmode=require"

then run the previous command.
